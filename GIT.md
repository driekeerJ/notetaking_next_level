# Empty commit
`git commit --allow-empty -m "chore: trigger build"`  
`git push`

# Branches
## Local branch
`git checkout my-branch-name`

## New local branch
`git checkout -b my-branch-name`

## Remote branch
`git checkout --track my-branch-name`

# Push
## Normal
`git push`

## New branch
`git push --set-upstream origin feature/5203734-simplify-real-estate`
